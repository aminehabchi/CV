# CIA Portfolio - Amine Habchi

A unique portfolio website styled as a classified CIA document/dossier.

## Features

- **CIA Document Theme** - Styled to look like a confidential government file
- **Fully Responsive** - Maintains the same layout on all screen sizes with proportional scaling
- **Custom Scrollbar** - Themed to match the document aesthetic
- **Professional Layout** - Two-column design with personal info, skills, projects, and contact details

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Courier Prime)

## File Structure

```
├── index.html          # Main HTML file
├── style.css           # Stylesheet with responsive design
├── ahabchi.jpg         # Profile photo
└── background.jpeg     # Document background texture
```

## Setup

1. Clone or download the repository
2. Make sure all files are in the same directory
3. Open `index.html` in your browser

## Customization

To customize for your own portfolio:

1. **Personal Info** - Edit the info rows in `index.html`
2. **Skills** - Update the favorites section with your technologies
3. **Projects** - Replace project links and descriptions
4. **Contact** - Update email, GitHub, LinkedIn, and phone number
5. **Images** - Replace `ahabchi.jpg` with your photo

## Responsive Design

The portfolio automatically scales for different screen sizes:
- **Desktop (900px+)** - Full-size layout
- **Tablet (768px)** - Proportionally scaled down
- **Mobile (480px)** - Further scaled for phones

All elements maintain the same layout structure across all devices.
